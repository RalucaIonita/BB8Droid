BB8 Arduino Droid
===============

	Final project for "Introduction to Robotics" course.
	
Components used
---------

Hardware

* Arduino Uno
* 2 breadboards
* L293D driver
* 2 DC motors with wheels
* Arduino prototyping board
* MP3 module
* MicroSD memory card
* 8 ohm speaker
* bluetooth module
* logical converter
* battery holder (4 x AA)
* 4 AA 1.5V batteries
* 1 9V battery
* wires
* 3D printed components
	
Software

+ Arduino IDE
+ Arduino Remote LITE (for Android)
+ Notepad++

Project

	This project contains three files:
	
* main.ino
* MotorFunctions.hpp
* MP3Functions.hpp
